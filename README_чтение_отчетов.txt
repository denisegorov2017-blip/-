Как система читает и обрабатывает отчеты
=====================================

Этот проект демонстрирует, как система читает реальные данные из Excel-файлов
и обрабатывает их для расчета коэффициентов усушки.

Созданные файлы:
-------------

1. examples\обработка_скумбрия_хк_верификация.py
   - Скрипт для обработки данных по "СКУМБРИЯ Х/К" с верификацией коэффициентов

2. examples\демонстрация_чтения_отчета.py
   - Демонстрация процесса чтения отчета на примере "БЫЧКИ ВЯЛЕНЫЕ"

3. examples\анализ_бычки_валеные.py
   - Детальный анализ данных по "БЫЧКИ ВЯЛЕНЫЕ" из реального файла

4. документация\как_система_читает_отчеты.md
   - Подробная документация о процессе чтения отчетов

Как работает система чтения:
------------------------

1. Загрузка файла:
   - Используется pandas.read_excel() для чтения Excel-файлов
   - Поддерживаются форматы .xls и .xlsx

2. Анализ структуры:
   - Поиск строк с номенклатурами рыбной продукции
   - Определение столбцов с числовыми данными

3. Извлечение данных:
   - Столбец 4: Начальный остаток
   - Столбец 6: Приход
   - Столбец 7: Расход
   - Столбец 8: Конечный остаток

4. Расчет усушки:
   - Теоретический баланс = Начальный + Приход - Расход
   - Фактическая усушка = Теоретический баланс - Конечный остаток

5. Обработка системой:
   - Подготовка данных для ShrinkageSystem
   - Расчет коэффициентов усушки
   - Верификация результатов

Запуск примеров:
-------------

Для запуска примеров выполните в командной строке:

python examples\обработка_скумбрия_хк_верификация.py
python examples\демонстрация_чтения_отчета.py
python examples\анализ_бычки_валеные.py

Результаты будут сохранены в папке "результаты".
