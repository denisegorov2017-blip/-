Рассчитай коэффициенты a, b и c для предварительной усушки по следующим данным:

1. Данные из Excel по номенклатуре [НАИМЕНОВАНИЕ НОМЕНКЛАТУРЫ] за период [ДД.ММ.ГГ] по [ДД.ММ.ГГ]
2. Не учитывай инвентаризацию [НОМЕР] от [ДД.ММ.ГГ] как начало периода (она выравнивает остатки)
3. Используй данные последующих инвентаризаций для определения фактической усушки
4. Применяй принцип FIFO (первая партия - первая продажа)
5. Учитывай, что вес одной единицы товара составляет [ВЕС] грамм

Формула для усушки: 
Потери(партия) = Σ[день=1..t] (Масса_на_начало_дня * a * b * exp(-b * день)) + c * Начальная_масса

Где:
- a: Макс. % потерь (0..1)
- b: Скорость усушки (день⁻¹)
- c: Мгновенные потери при приёмке
- t: Дни хранения
- Масса на день = Начальная масса - продажи до этого дня

Процесс расчета:
1. Вставь данные из Excel в формате: Дата | Документ | Количество (приход/расход) | Остаток
2. Для каждой партии определи время хранения с учетом FIFO
3. Распредели усушку из инвентаризаций по партиям
4. Рассчитай коэффициенты a, b, c методом наименьших квадратов
5. Проверь точность модели (сравни с линейной моделью)
6. Выведи итоговые коэффициенты и их обоснование
7. Укажи процент точности по сравнению с другими методами

Формат вывода: |[НАИМЕНОВАНИЕ НОМЕНКЛАТУРЫ]|a = [ЗНАЧЕНИЕ]|b = [ЗНАЧЕНИЕ] (день⁻¹)|c = [ЗНАЧЕНИЕ]|Вес_единицы = [ЗНАЧЕНИЕ] (граммы)|Точность = [ЗНАЧЕНИЕ]%|Дата_расчета = [ДД.ММ.ГГ]|Примечание = [ОПЦИОНАЛЬНОЕ ПРИМЕЧАНИЕ]|

Примечание: Для расчета используй только данные после первой инвентаризации периода, которая выравнивает остатки.
