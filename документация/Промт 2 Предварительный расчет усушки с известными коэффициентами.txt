Рассчитай предварительную усушку по следующим данным:

1. Данные из Excel по номенклатуре [НАИМЕНОВАНИЕ НОМЕНКЛАТУРЫ] за период [ДД.ММ.ГГ] по [ДД.ММ.ГГ]
2. Коэффициенты для этой номенклатуры (из файла коэффициентов):
   - a = [ЗНАЧЕНИЕ]
   - b = [ЗНАЧЕНИЕ] день⁻¹
   - c = [ЗНАЧЕНИЕ]
3. Применяй принцип FIFO (первая партия - первая продажа)
4. Учитывай, что вес одной единицы товара составляет [ВЕС] грамм

Формула для усушки: 
Потери(партия) = Σ[день=1..t] (Масса_на_начало_дня * a * b * exp(-b * день)) + c * Начальная_масса

Где:
- a: Макс. % потерь (0..1)
- b: Скорость усушки (день⁻¹)
- c: Мгновенные потери при приёмке
- t: Дни хранения
- Масса на день = Начальная масса - продажи до этого дня

Процесс расчета:
1. Вставь данные из Excel в формате: Дата | Документ | Количество (приход/расход) | Остаток
2. Для каждой партии определи время хранения с учетом FIFO
3. Для каждой партии рассчитай усушку по дням с учетом уменьшения массы партии по мере продаж
   (Пример: если остаток партии 10 кг не расходовался 2 дня, то усушка считается от 10 кг; 
    на 3-й день продалось 2 кг, то со следующего дня расчет усушки ведется от 8 кг)
4. Выведи итоговую усушку и распределение по партиям
5. Сравни результаты с линейной моделью (при необходимости)

Примечание: Расчет должен учитывать изменение массы партии каждый день на основе фактических продаж.
